# Smoke-тести веб-додатку

## 1. Користувачі

1. **Реєстрація користувача**  
   Кроки: Відкрити сторінку реєстрації → Ввести валідні дані → Натиснути "Реєстрація"  
   Очікуваний результат: Користувач зареєстрований, перенаправлення на особистий кабінет, сповіщення про успіх  
   Результат: ✅  
   Коментар: Успішно, реєстрація проходить без помилок

2. **Вхід користувача**  
   Кроки: Відкрити сторінку входу → Ввести валідні дані → Натиснути "Вхід"  
   Очікуваний результат: Користувач залогінений, бачить особистий кабінет  
   Результат: ✅  
   Коментар: Вхід працює коректно, користувач успішно залогінений

3. **Вихід користувача**  
   Кроки: Натиснути кнопку "Вихід"  
   Очікуваний результат: Користувач виходить, перенаправлення на головну сторінку  
   Результат: ✅  
   Коментар: Вихід виконано успішно, перенаправлення працює

---

## 2. Новини

1. **Перегляд списку новин**  
   Кроки: Відкрити сторінку новин  
   Очікуваний результат: Відображаються всі останні новини, картки коректні  
   Результат: ✅  
   Коментар: Список новин відображається правильно, всі дані на місці

2. **Створення новини (адмін)**  
   Кроки: Увійти як адміністратор → Відкрити CRUD новин → Додати новину  
   Очікуваний результат: Новина створена, відображається в списку  
   Результат: ✅  
   Коментар: Новина успішно створена та відображається в списку

3. **Редагування новини (адмін)**  
   Кроки: Натиснути "Редагувати" на новині → Змінити текст/заголовок → Зберегти  
   Очікуваний результат: Новина оновлена, зміни відображаються  
   Результат: ✅  
   Коментар: Редагування працює, зміни застосовані успішно

4. **Видалення новини (адмін)**  
   Кроки: Натиснути "Видалити" на новині  
   Очікуваний результат: Новина видалена, більше не відображається в списку  
   Результат: ✅  
   Коментар: Новина видалена без проблем, список оновлено

## 4. Онлайн-звернення

1. **Надсилання звернення**  
   Кроки: Відкрити форму звернення → Заповнити всі поля → Натиснути "Відправити"  
   Очікуваний результат: Звернення створено, сповіщення про успіх  
   Результат: ✅  
   Коментар: Форма працює коректно, звернення надіслано

2. **Перегляд звернень (користувач)**  
   Кроки: Увійти у кабінет → Відкрити список звернень  
   Очікуваний результат: Відображаються всі власні звернення  
   Результат: ✅  
   Коментар: Всі звернення користувача відображаються

3. **Перегляд звернень (адмін)**  
   Кроки: Увійти як адміністратор → Відкрити список звернень  
   Очікуваний результат: Відображаються всі звернення користувачів, можна фільтрувати  
   Результат: ✅  
   Коментар: Відображення та фільтрація працюють коректно

   Очікуваний результат: Список новин коректно відображається  
   Результат: ✅  
   Коментар: API працює стабільно, дані відображаються

2. **Перевірка API сервісів**  
   Кроки: Відкрити сторінку сервісів → Переконатися, що дані завантажуються з API  
   Очікуваний результат: Сервіси відображаються коректно  
   Результат: ✅  
   Коментар: API сервісів працює без помилок

3. **Перевірка CORS**

   Очікуваний результат: Запит успішний, помилок CORS немає  
   Результат: ✅  
   Коментар: CORS налаштований правильно, запити проходять
